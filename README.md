## About

> Grid prototype


## Uses

- [Node.js](https://nodejs.org/) v20.9.0 or later


## Project setup

```bash
npm install
```

### Serve with hot reload at localhost:5173

```bash
npm run dev
```

### Run the linting

```bash
npm run lint
```

### Compiles and minifies for production

```bash
npm run build
```
This command generates the `dist` folder with the final production files.
